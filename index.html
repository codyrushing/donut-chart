Data

This is designed to be used with progress-style donut charts. In addition to your array of values, you will need a total key.

        var data = {
            total: 70000, 
            values: [
                { n: 40125, color: "#6dab42" }, 
                { n: 10000, color: "#ff0000" }, 
                { n: 5000, color: "#0000ff" }, 
                { n: 5000, color: "#ffff00" }, 
                { n: 3000, color: "#f92a39" }
            ]
        };
